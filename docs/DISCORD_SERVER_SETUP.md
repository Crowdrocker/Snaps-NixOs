## Moderation Setup

### AutoMod Rules

**Rule 1: Spam Prevention**
- Block repeated messages
- Block excessive mentions
- Block excessive emojis

**Rule 2: Link Protection**
- Block suspicious links
- Allow trusted domains

**Rule 3: Caps Lock**
- Block excessive caps

**Rule 4: Bad Words**
- Block offensive language

### Moderation Bots

**Recommended Bots:**

1. **MEE6**
   - Leveling system
   - Auto-moderation
   - Custom commands

2. **Dyno**
   - Moderation tools
   - Auto-roles
   - Announcements

3. **Streamcord** (or similar)
   - Twitch stream notifications
   - Auto-post when live

4. **Statbot**
   - Server statistics
   - Member tracking
